To create a login form we use three main tags 1.form 2.label 3.input Input is used to specify the type of data Types of input 1.text → when the input is text 2.password → when the input is collection of characters and numbers and special characters 3.submit →used for submitting the form Attributes of input 1.name :used when user and password is same 2.id :id is used in input to link label and output 3.required : is used to mandatory fill the data
